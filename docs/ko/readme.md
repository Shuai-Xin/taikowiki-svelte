# Taikowiki-svelte 

이 프로젝트는 태고의 달인을 위한 위키 프로젝트입니다. 오픈소스로 개발되며, 모든 기여는 환영입니다. 코드 뿐 아니라 번역, 디자인 등도 기여하실 수 있습니다.

## 기여하기
